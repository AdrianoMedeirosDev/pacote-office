Principais Atalhos




ATALHOS WORD
Atalho Função
Ctrl + N Cria um novo documento em branco.
Ctrl + O Abre um documento existente.
Ctrl + S Salva o documento atual.
Ctrl + Z Desfaz a última ação realizada.
Ctrl + Y Refaz a última ação desfeita.
Ctrl + C Copia o texto ou objeto selecionado.
Ctrl + X Recorta o texto ou objeto selecionado.
Ctrl + V Cole o texto ou objeto copiado ou recortado.
Ctrl + B Aplica o negrito ao texto selecionado.
Ctrl + I Aplica o itálico ao texto selecionado.
