Principais Atalhos




ATALHOS POWER POINT
Atalho Função
Ctrl + N Cria uma nova apresentação em branco.
Ctrl + O Abre uma apresentação existente.
Ctrl + S Salva a apresentação atual.
Ctrl + Z Desfaz a última ação realizada.
Ctrl + Y Refaz a última ação desfeita.
Ctrl + C Copia o objeto ou texto selecionado.
Ctrl + X Recorta o objeto ou texto selecionado.
Ctrl + V Cole o objeto ou texto copiado ou recortado.
Ctrl + P Abre a caixa de diálogo de impressão para imprimir a apresentação.
F5 Inicia a apresentação do slide atual a partir do início.
