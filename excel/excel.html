Principais Atalhos




ATALHOS POWER POINT
Atalho Função
Ctrl + N Cria uma nova pasta de trabalho em branco.
Ctrl + O Abre uma pasta de trabalho existente.
Ctrl + S Salva a pasta de trabalho atual.
Ctrl + Z Desfaz a última ação realizada.
Ctrl + Y Refaz a última ação desfeita.
Ctrl + C Copia as células ou objetos selecionados.
Ctrl + X Recorta as células ou objetos selecionados.
Ctrl + V Cole as células ou objetos copiados ou recortados.
Ctrl + P Abre a caixa de diálogo de impressão para imprimir a planilha.
Ctrl + F Abre a caixa de diálogo "Localizar" para procurar por um valor específico na planilha.
